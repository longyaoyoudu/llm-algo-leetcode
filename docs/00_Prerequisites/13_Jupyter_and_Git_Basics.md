# 13. Jupyter and Git Basics Practice | Jupyter and Git Basics - 练习

**难度：** Easy | **标签：** `Notebook`, `Git`, `Version Control`, `Workflow` | **目标人群：** Chapter 0 入门学习者

## 学习目标

- 学会识别 notebook 里的 code / markdown cell
- 理解哪些路径通常不应提交到仓库
- 为后续的 Git 工作流和提交清单打底

## 核心练习

- `count_code_cells`
- `list_markdown_headings`
- `is_ephemeral_artifact`
- `summarize_notebook`

## 练习提示

- notebook 元数据不复杂时，优先用最简单的结构解析
- 把缓存、临时目录和备份文件视为不该提交的对象
- 提交前先看 git status，再决定 git add 范围
